Политика конфиденциальности

Кратко

  • Приложение работает офлайн.
  • Регистрация пользователя не требуется.

  • Разработчик не получает, не продаёт и не передаёт пользовательские данные третьим лицам.

1. Общие положения

Настоящая Политика конфиденциальности описывает, какие данные используются приложением «Корзина КБЖУ», как они хранятся и каким образом пользователь может управлять своими данными.

2. Назначение приложения

Приложение предназначено для планирования продуктовой корзины, расчёта калорий, белков, жиров и углеводов, а также анализа соответствия корзины целям пользователя.

3. Возрастные ограничения

Приложение предназначено для пользователей старше 18 лет. Приложение не предназначено для детей и подростков.

4. Какие данные вводит пользователь

Пользователь может самостоятельно вводить следующие данные: пол, возраст, рост, вес, уровень активности, цель питания, рассчитанные или заданные вручную показатели КБЖУ, список продуктов в корзине, пользовательские продукты и их пищевую ценность.

5. Какие данные приложение не собирает

Приложение не собирает паспортные данные, банковские данные, контакты, геолокацию, фотографии, данные социальных сетей и другую информацию, не требующуюся для работы приложения.

6. Где хранятся данные

Все пользовательские данные хранятся локально на устройстве пользователя. Приложение не передаёт эти данные на серверы разработчика, не использует облачную синхронизацию и не требует регистрации.

7. Передача данных третьим лицам

Разработчик не продаёт, не передаёт и не предоставляет пользовательские данные третьим лицам.

8. Реклама, аналитика и сторонние сервисы

В текущей версии приложение не использует рекламу, Firebase, аналитические сервисы, подписки, встроенные покупки и внешние серверы.

9. Разрешения устройства

Приложение запрашивает только те разрешения, которые необходимы для его работы. Если в будущих версиях будут добавлены функции, требующие дополнительных разрешений, пользователь будет уведомлён стандартными средствами операционной системы.

10. Безопасность данных

Поскольку данные хранятся локально, безопасность информации зависит в том числе от безопасности устройства пользователя. Рекомендуется использовать блокировку экрана и не передавать устройство третьим лицам.

11. Удаление данных

Пользователь может удалить данные внутри приложения, очистить корзину, удалить цель или удалить приложение. После удаления приложения локальные данные удаляются средствами операционной системы.

12. Медицинская информация

Приложение не является медицинским изделием, не предназначено для диагностики, лечения или профилактики заболеваний и не заменяет консультацию врача, диетолога или другого специалиста.

13. Изменения политики

Разработчик может обновлять настоящую Политику конфиденциальности. При существенных изменениях будет обновлена дата вступления документа в силу.
